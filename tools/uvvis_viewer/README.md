# UV-Vis Viewer

## 功能概述

UV-Vis Viewer 是一个基于 PyQt6 和 Matplotlib 的光谱数据可视化工具，用于查看和分析 UV-Vis 光谱数据。它支持以下核心功能：

- 加载和显示 UV-Vis 光谱数据
- 计算和显示吸光度
- 可选的光谱归一化功能
- 峰值查找和标记
- 自定义波长范围显示
- 十字光标功能，显示当前鼠标位置的波长和吸光度值
- 保存光谱图像为 PNG 格式

## 安装与配置

### 前提条件

- Python 3.7 或更高版本
- 以下 Python 包：
  - PyQt6
  - matplotlib
  - numpy
  - pandas
  - scipy
  - a304data（项目内部包）

### 安装步骤

1. 克隆或下载项目代码到本地
2. 确保已安装所有依赖包
3. 进入 `tools/uvvis_viewer` 目录

### 运行方式

直接运行 `main.py` 文件：

```bash
python main.py
```

## 使用指南

### 基本操作流程

1. **打开数据目录**：点击 "选择数据目录" 按钮，选择包含 UV-Vis 数据文件的目录
2. **选择光谱**：在 "光谱" 下拉菜单中选择要查看的光谱
3. **调整显示范围**：通过 "wl_min" 和 "wl_max" 输入框设置波长显示范围
4. **查找峰值**：勾选 "查找峰值" 复选框，可调整 height、prom 和 dist 参数来优化峰值查找
5. **归一化控制**：使用 "归一化" 复选框控制是否对光谱进行归一化
6. **保存图像**：点击 "保存 PNG" 按钮将当前显示的光谱保存为 PNG 格式

### 主要功能说明

#### 归一化功能

归一化功能会将所有光谱的吸光度值除以整个数据集中的最大吸光度值，使得所有光谱的吸光度最大值为 1。这有助于比较不同光谱的相对强度。

#### 峰值查找

峰值查找功能使用 `scipy.signal.find_peaks` 算法来识别光谱中的峰值，并在图上标记出来。您可以通过调整以下参数来优化峰值查找：

- **height**：峰值的最小高度
- **prom**：峰值的最小显著性
- **dist**：峰值之间的最小距离

#### 十字光标

当鼠标在光谱图上移动时，会显示十字光标，并在状态栏中显示当前位置的波长和吸光度值。

## 目录结构

```
tools/uvvis_viewer/
├── README.md          # 本文档
├── build.bat          # 打包脚本
├── controller.py      # 控制器，处理数据加载和处理
```

### 文件说明

- **controller.py**：包含 `UVVisController` 类，负责加载数据集、计算吸光度和处理峰值查找。
- **window.py**：包含 `MainWindow` 类，负责创建和管理用户界面。
- **main.py**：应用程序入口点，创建并显示主窗口。
- **build.bat**：用于将应用程序打包为可执行文件的脚本。

## 打包指南

要将 UV-Vis Viewer 打包为可执行文件，请运行 `build.bat` 脚本：

```bash
build.bat
```

默认情况下，脚本会生成一个包含所有依赖项的目录（`--onedir` 模式），并使用 `harumasa.ico` 作为应用程序图标。

## 常见问题解答 (FAQ)

### Q: 为什么打开数据目录后没有显示光谱？
A: 请确保目录中包含正确格式的 UV-Vis 数据文件。对于 FX2000 设备，文件名应包含 'FX2000_FX2K243001' 标签，且应有一个背景文件（文件名包含 'bg' 或 'background'）。

### Q: 为什么归一化功能没有生效？
A: 归一化功能只在加载数据时应用。如果您在加载数据后更改了归一化设置，请等待应用程序重新加载数据。

### Q: 如何调整峰值查找的灵敏度？
A: 您可以通过调整 height、prom 和 dist 参数来优化峰值查找。较低的 height 和 prom 值会检测到更多的峰值，而较大的 dist 值会减少检测到的峰值数量。

### Q: 为什么保存的图像质量不高？
A: 默认情况下，保存的图像使用 300 DPI 的分辨率。如果需要更高质量的图像，您可以修改 `save_figure` 方法中的 `dpi` 参数。
